# wp1101

## Reference
Apple calculator

## Usage
1. 沒支援先乘除後加減,但可以連續加減乘除
2. mc: store the current number
3. mr: recall the stored number
4. sci: change to scientific notation mode
5. root: output = x^(1/2)
6. sqrt: output = x^2
7. 1/x: output = 1/x
8. del: do backspace
9. AC: All clear
10. +/-: multiply by -1


## Error
1. Divided by 0
2. Not a real Number
